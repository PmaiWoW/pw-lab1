Acesso via HTTP a uma página Web

Nesta secção, foi-nos passada informação compactada de como um Web browser comunica com um servidor Web através do protocolo HTTP.

Protocolo de troca de mensagens HTTP

Muito sucintamente, o cliente (browser) envia uma mensagem HTTP GET, de forma a pedir uma página Web ao servidor.
Em torno devolve os conteúdos necessários para que o cliente possa exibir visualmente o mesmo.
Nota: Uma forma de analisar os conteúdos que o servidor devolve a um cliente será abordada na subsecção seguinte.

O cliente pode também enviar dados ao servido através de uma mensagem HTTP POST, por exemplo de um formulário preenchido.
